## 🗺️ これは何？

Ingress Intel Map（[intel.ingress.com](https://intel.ingress.com)）を開いた状態で COMM ALL のログやマップ上のイベントをリアルタイムに自動検知し、AI がニコニコ動画風のディープなコメントを生成して地図上に流す拡張スクリプトです。

- **GPUアクセラレーションによる超軽量動作 (v1.2.0~)**: CSS Variablesの廃止と静的キーフレーム（`translate3d`）への刷新により、大量のコメントが流れる際でもプチフリーズを一切発生させず、非常に滑らかなスクロールを実現しています。
- **リアルタイム戦況・チャット反応**: ポータル名・地名・プレイヤー名、ポータルのキャプチャ、中和、コントロールフィールド（CF）の形成/崩壊、リンク確立などを自動検知してAIコメントがトリガーされます。
- **音声読み上げ機能 (v1.2.0 TTS競合バグ解消)**: AIコメントを合成音声（Web Speech API）で読み上げます。一般コメントはゆっくりボイス風、NPC陣営（MACHINA）は英語の不気味な専用声色で再生されます。コメント消滅を正確に監視することで、連続発話時でも音声が途切れることなく最後まで自然に再生されます。
- **3つの最新AIモデルに対応**: OpenAI / Claude / Gemini（チェックしたAPIを OpenAI → Claude → Gemini の固定順で使用し、エラー時は次のAPIへ自動フォールバック）。無料枠のあるGemini（Google AI Studio）だけでも十分に動作します。
- **陣営バイアス＆対人的陰謀論 (v1.2.0プロンプト高度化)**: エージェント文化（チャリチャリ、徒歩、車載、警察の職質など）を豊富に網羅。「あの深夜のデプロイは陽動作戦か？」「二重スパイを企んでいるのでは」といった、特定エージェントの企みに対するプレイヤー特有の根拠のない邪推や陰謀論コメントが低確率で出現します。
- **時間帯ペルソナ (v1.7.0~)**: 現在の時刻と曜日をAIが認識し、深夜なら「深夜徘徊・職質リスク」、通勤時間帯なら「出勤前ワンデプロイ」、週末なら「遠征・アノマリー」といった、その時間ならではのエージェントの生活感がコメントに自然に混ざります。

## 📋 必要なもの

- PC版 Chrome または Microsoft Edge
- 拡張機能 **Tampermonkey**
- **AIのAPIキー**（以下のいずれか1つ以上）

| AI | 推奨モデル | APIキー取得先 |
|---|---|---|
| **OpenAI** | `gpt-4o-mini` | [platform.openai.com/api-keys](https://platform.openai.com/api-keys) |
| **Claude** | `claude-haiku-4-5` | [console.anthropic.com/settings/keys](https://console.anthropic.com/settings/keys) |
| **Gemini** | `gemini-2.5-flash` | [aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey) |

> 💡 **Geminiは無料枠あり**  
> Google AI StudioのGemini APIは無料枠での利用が可能です。まずはGeminiから試すのがお勧めです。

## ⚙️ インストールと初期設定手順

1. **Tampermonkeyをインストール**  
   お使いのブラウザに [`Tampermonkey` 拡張機能](https://www.tampermonkey.net/)を追加します。
2. **スクリプトを追加**  
   👉 **[ここをクリックしてスクリプトをインストール](https://raw.githubusercontent.com/MikanRobot/nico-intelmap/main/ingress_niconico_comments.user.js)**  
   *※Tampermonkeyのインストール画面が開くので「インストール」を押してください。（手動の場合は、当リポジトリの `ingress_niconico_comments.user.js` の中身をコピペして保存します）*
3. **Ingress Intel Mapを開く**  
   [intel.ingress.com](https://intel.ingress.com) にアクセスします。
   - **初期状態でパネルは折りたたまれています**。画面右下に控えめに表示される「🎌 ニコニコインテルマップ ▲」タイトルバーのクリック、または**タイトルバー部分のダブルクリック**によって展開・開閉が可能です（インテルマップの視認性を邪魔しません）。
   - コントロールパネルは**ドラッグ移動が可能**です。タイトルバーを掴んでお好みの位置へスムーズに配置できます（クリック時に配置がワープするバグは修正済みです）。
4. **APIキーを入力＆起動時自動検証 (v1.2.0~)**  
   パネル内の「API設定」タブを開き、取得済みのAPIキーを入力します。
   - **自動一括検証**: キー入力後、または次回起動時には、保存済みのAPIキーが自動的にバックグラウンドで接続検証されます。
   - **概要表示領域**: API設定タブの最下部に、現在どのモデルが「読み込み完了」していて利用可能な状態にあるかが、リアルタイムで動的に連動表示されます。
5. **コメントの受信**  
   COMM ALL またはマップ上にイベントの動きがあると、AIが内容を自動で読み取ってコメントを生成し、画面右から左へ流します。

## 🎛️ パネルの使い方（v1.8.0でUIを一新）

### ヘッダー & ステータスバー（常時表示）
- **稼働状態ドット**: パネル左肩の●で状態が一目で分かります（🟢稼働中 / 🟠AI生成上限に到達 / 🔴プラグイン無効・APIキー未設定）。
- **電源トグル**: ヘッダー右の⬤スイッチでプラグイン全体（コメント描画・API連携）のON/OFFを切り替えます。
- **ステータスバー**: 折りたたんだ状態でも `🔄 AI残量 ・ 💰 本日コスト ・ ▶ 直近使用API` を常時表示します。バーをクリックするとパネルが開きます。

### 🖱️ ショートカット操作
- **タイトルバーのダブルクリック / ステータスバーのクリック**: パネルの開閉。

### 「基本」タブ
- **コメント表示 → 1回の生成数**: 1回の生成で流れるコメント数を指定（1〜100 / 初期設定: 7）。
- **音声読み上げ**: カード見出しのトグルでON/OFF（一般コメントは日本語ボイス / MACHINAは不気味な英語ボイス）。OFF時は下の「読み上げ方式」が減光されます。
- **読み上げ方式**: 「ゆっくり」（早口・高ピッチ）と「普通」（標準速度・標準ピッチ）をセグメント切り替え。

### 「AI / API」タブ
- **AI生成予算**: AIコメント生成の残り回数（例: `3 / 5`）とプログレスバー、**推定コスト（本日／今回セッション）**を表示。上限到達後も「**＋5回延長**」ボタンでその場で再開できます（ボタンは上限到達時に点滅して知らせます）。
- **APIキー**: OpenAI / Claude / Gemini のキーをカード単位で管理。各カードのトグルで個別にON/OFF、検証状態をアイコン（✅OK / ❌エラー・未設定 / ⏳検証中）で表示します。チェック済みのAPIを **OpenAI → Claude → Gemini の固定順**で使用し、エラー時は次のAPIへ自動フォールバックします。

### 「ログ」タブ
- **デバッグログ**: 接続中のAIからのレスポンス、パケット取得ログ、トークン消費量と概算コストなどがリアルタイムで流れます（📋コピーボタンでログ全文をクリップボードへコピー可能）。

## 💬 コメントの種類

| 種類 | 色 | 説明 |
|---|---|---|
| 一般エージェント | 白 | ログに対する自然な反応。稀に「アノマリーの仕込みか？」といった戦術的・対人的な邪推（陰謀論）も混ざる。 |
| Ingressガチ勢 | 白 | MU・CF・リンク戦略、ポータルキーの防衛リチャージなどを冷静に分析する。 |
| RESバイアス | 青 | レジスタンス陣営を贔屓し、ENLを煽るコメント。毎回の生成で必ず1件以上出現。RES有利なログがあれば内容に絡めて反応する。 |
| ENLバイアス | 緑 | エンライテンド陣営を贔屓し、RESを煽るコメント。毎回の生成で必ず1件以上出現。ENL有利なログがあれば内容に絡めて反応する。 |
| MACHINA | 赤 | 謎のNPC陣営「MACHINA」の侵食時に極低確率で出現する不気味な半角英語コメント。文字が乱れるグリッチエフェクト付き。 |

## ⚠️ 注意事項

- 本プラグインは Niantic/Ingress の公式とは一切関係ありません。
- APIキーはブラウザのローカルストレージ（Tampermonkeyの管理領域）に安全に保存されます。共用PCでは利用しないでください。
- AIの利用料金はご自身のAPIアカウントに対して発生します。
- **API利用料の暴走防止（v1.6.0~）**: Intel Mapは内部で定期的にログを自動読み込みします。API課金の垂れ流しを防ぐため、新規ログを伴う自動読み込みが**5回**に達すると、それ以降のAIコメント生成（API呼び出し）は自動停止します。メインタブの「**＋5回延長**」ボタンで再開でき、ページを再読み込みするとカウントはリセットされます。
- ブラウザの自動再生ブロック（Autoplay Policy）の仕様上、**画面を開いた直後は音声が再生されません。** マップ上を1度クリックやドラッグ操作をすることで音声エンジンのロックが解除され、以降のコメントから自動読み上げが開始されます（最初のアクションまではテキスト表示のみが進行します）。
- **DOMリークセーフティネット (v1.2.0~)**: 万が一ブラウザがバックグラウンドに回るなどしてアニメーション終了イベントが発火しなかった場合でも、アニメーション終了予定時刻の2秒後に発火するセーフティネットタイマーによってコメント要素は確実にDOMから自動消滅し、メモリリークを防ぎます。

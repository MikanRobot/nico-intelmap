🗺️ これは何？

Ingress Intel Map（intel.ingress.com）を開いた状態で、COMM ALLのログを読み取り、AIがニコニコ動画風のコメントを生成して地図上に流します。

  • ポータル名・地名・プレイヤー名を拾ってリアルタイムにコメント
  • OpenAI / Claude / Gemini の3つのAPIに対応（自動ランダム選択）
  • 陣営バイアスコメント（RES=青、ENL=緑）も稀に出現

📋 必要なもの

  • PC版 Chrome または Firefox
  • Tampermonkey 拡張機能（ブラウザにインストール済みであること）
  • AIのAPIキー（以下のいずれか1つ以上）
AI モデル APIキー取得先
OpenAI gpt-4o-mini platform.openai.com/api-keys
Claude claude-haiku-4-5 console.anthropic.com/settings/keys
Gemini gemini-2.0-flash aistudio.google.com/app/apikey
💡 Geminiは無料枠あり Google AI StudioのGemini APIは無料枠での利用が可能です（2025年3月時点）。まずはGeminiから試すのがお勧めです。

⚙️ インストール手順

1
Tampermonkeyをインストール

お使いのブラウザの拡張機能ストアから「Tampermonkey」を検索しインストールしてください。

2
スクリプトをTampermonkeyに追加

Tampermonkeyのダッシュボード →「新規スクリプト」→ ingress_niconico_comments.user.js の内容を貼り付けて保存します。

3
Ingress Intel Mapを開く

intel.ingress.com にアクセスします。右上に「ニコニコインテルマップ」パネルが出れば成功です。

4
「API設定」タブでAPIキーを入力

パネルの「API設定」タブを開き、取得済みのAPIキーを入力します。フォーカスを外すと自動で検証され ✅ API Key OK と表示されます。

5
COMMALLを眺めてコメントが流れるのを待つ

地図上でゲームのアクティビティが検知されると、約30秒間隔でAIがコメントを生成して地図上に流します。手動で試したい場合は「今すぐ流す」ボタンを押してください。

🎛️ パネルの使い方

メインタブ

  • プラグイン有効: チェックを外すとコメント表示・AI処理をすべて停止します
  • コメント数: 1回のAI呼び出しで生成するコメント数（1〜100）
  • デバッグ表示: AI呼び出し状況・エラーログをリアルタイムで確認できます
  • 今すぐ流す: 手動でAIコメント生成を実行します

API設定タブ

  • OpenAI / Claude / Gemini のAPIキーを入力＆自動保存
  • 複数設定した場合は呼び出しごとにランダム選択されます
  • キー入力後にフォーカスを外すと自動で疎通確認します

操作

  • パネルはドラッグして好きな場所に移動できます
  • タイトルバーの ▼ ボタンでパネルを折りたたむことができます

💬 コメントの種類

一般視聴者（白）
全コメントの85%以上。ポータル名や地名を拾った自然な反応。稀に陰謀論や深読みコメントも。
Ingressガチ勢（白）
5%以下。MU・CF・リンク戦略など専門的な視点で冷静に分析。
レジスタンス陣営バイアス（青）
3%以下。RES有利なアクションがあった場合のみ出現。
エンライテンド陣営バイアス（緑）
3%以下。ENL有利なアクションがあった場合のみ出現。
